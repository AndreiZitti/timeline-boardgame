## Games

| Game | Description |
|------|-------------|
| **Hot Take** | Players get secret numbers 1-100 and describe where they stand on a spectrum |
| **Like Minded** | Wavelength-style psychic guessing game with spectrums |
| **Secret Hitler** | Social deduction game (coming soon) |

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Next.js 16, React 19 |
| Database | Supabase (realtime) |
| Styling | CSS Variables, Framer Motion |
| Language | TypeScript, JavaScript |

## Getting Started

```bash
# Clone
git clone https://github.com/AndreiZitti/games.zitti.git
cd games.zitti

# Install
npm install

# Environment
cp .env.example .env.local
# Add your Supabase keys

# Run
npm run dev
```

Open http://localhost:3000

## Environment Variables

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```
